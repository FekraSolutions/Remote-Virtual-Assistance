### Patient Admission
Easily register a new patient or create an invoice from a single, simple window. It displays and stores all patient details in the admission log. After generating an invoice, you can directly book the patient’s sessions or appointments.

### Sessions Schedule
The blue schedule is a smart tool that shows you the day you’re booking for, displaying the data of that day in the gray table to help you find free slots. Finally, click “Save” to record the data in the green table (the session database) easily.

### Records
All patient admissions are stored in the Records page. You can filter, print, or edit entries as you wish.

### Invoices
You can search invoices by ID number or phone number and print them instantly.

### Send Invoice via WhatsApp
A simple mechanism allows you to send any invoice directly to the client on WhatsApp.

### Payroll Report
Automatic reports, simple data entry, and instant printing of all payroll information.

### Expenses Log
An easy way to record and view expense reports.

### Patient Files
Ready-to-print reports with a smart and simple database that includes all medical files related to each patient.  
You can also view all the patient’s sessions and invoices from the same page, and control the page layout and font size to match any print form or template.

### Profit and Loss Report
Just select the start and end dates of the desired period. The result automatically displays the profit statement, a revenue analysis chart, and a detailed income and expense breakdown. Ready to print.

**What technology is used to develop the app?**  
The application is based on a single Google Sheets file — not linked to any other files. It contains its own independent script code and Web App Deployments connected to:  
1. The app’s own features  
2. Tax authority APIs / compatible systems  

**What are the system requirements for running ClinicERP?**  
Only an internet connection and any web browser! **ClinicERP** is designed to be extremely lightweight and practical — no powerful hardware required. It runs very fast even on mid-range or below-average devices.  
Any old PC or laptop can run it without issues.
